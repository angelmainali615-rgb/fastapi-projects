# FastAPI Projects

This repository contains Python projects built using **FastAPI**, a modern, fast web framework for building APIs with Python.

## Topics Covered
- Creating RESTful APIs
- Routing and path parameters
- Query parameters and request bodies
- Dependency injection
- Using Pydantic for data validation
- Running FastAPI with Uvicorn
- Example projects demonstrating CRUD operations

## Purpose
These projects help practice building real-world APIs in Python and provide hands-on experience with web development using FastAPI.

## How to Run
1. Make sure Python and FastAPI are installed.
2. Install dependencies: `pip install fastapi uvicorn`
3. Run the API: `uvicorn main:app --reload`
4. Access the API in browser at `http://127.0.0.1:8000/`
